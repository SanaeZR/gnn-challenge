# Dataset Notes

This dataset is synthetic and designed for inductive node classification under class imbalance.

Files:
- train.csv: features + labels for the training graph
- test.csv: features for the test graph
- train_edges.csv: edges for the training graph
- test_edges.csv: edges for the test graph
- test_labels.csv: hidden labels (organizers only)

Graphs are undirected and use separate node ID ranges per split.
